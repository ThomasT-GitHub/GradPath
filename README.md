# GradPath

Your personalized academic planning tool for UCF students to help navigate their educational journey with confidence and clarity.

## About

GradPath is a web application designed for University of Central Florida students to plan their degree path, track academic progress, and ensure timely graduation.

## Tech Stack

- **Frontend Framework**: React 19 with TypeScript
- **Build Tool**: Vite 7
- **Styling**: Tailwind CSS 4
- **Linting**: ESLint with TypeScript support

## Getting Started

### Prerequisites

- Node.js v20.19.0 or higher (or v22.12.0+)
- npm

### Installation

```bash
# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev
```

### Build

```bash
# Build for production
npm run build
```

### Preview Production Build

```bash
# Preview production build locally
npm run preview
```

### Linting

```bash
# Run ESLint
npm run lint
```
